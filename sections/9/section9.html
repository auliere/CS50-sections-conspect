Вступ

Це передостанній семінар. У цьому семінарі мова піде про мову Javascript.

Javascript

Javascript - це мова програмування, що створена для виконання на стороні клієнта (на відміну від PHP, призначеного для виконання на стороні сервера). Це означає, що код Javascript виконується на комп'ютері користувача тоді, коли він завантажує сторінку.

Синтаксис Javascript досить схожий на синтаксис PHP або C. Javascript - це динамічно типізована мова, типи усіх змінних визначаються під час роботи програми.

Hello, world!

Нижче наведено приклад сторінки, в яку включено простий Javascript-файл, і вміст цього файлу.

index.html:

<!DOCTYPE html>	
<html>	
  <head>	
    <script type="text/javascript" src="hello.js"></script>	
    <title>Hello, world!</title>	
  </head>	
  <body>	
    Body HTML here
  </body>	
</html>

hello.js:

alert("Hello, world!");

Якщо відкрити у браузері сторінку index.html, то побачимо наступне:

Зверніть увагу, як ми включили в html-сторінку javascript-файл:

<script type="text/javascript" src="hello.js"></script>

При завантаженні сторінки підключений файл автоматично виконується браузером. Функція alert відповідає за показ віконця з текстовим повідомленням.

Оголошення змінних

Локальні змінні у javascript оголошують наступним чином:

var <ім'я змінної> = <значення змінної>

Глобальні змінні оголошують так само, але без слова var:

<ім'я змінної> = <значення змінної>

Наприклад:

s = "CS50"; //глобальна змінна
var n = 3.14; //локальна змінна
var b = true; //локальна змінна

Щоб змінити тип змінної, треба просто присвоїти їй нове значення:

b = "make b a string";

Цикли

Синтаксис циклів у Javascript аналогічний синтаксису C:

for(/* ініціалізація */; /* умова */; /* дія */)
{
  /* код */
}

while(/* умова */)
{
  /* код */
}

do {
  /* код */
} while (/* умова */);

Оголошення і виклик функцій

Синтаксис оголошення функцій також схожий на С:

function sum(x,y)
{
  return x + y;
}

У Javascript є іще один варіант оголошення функцій, за якого ми фактично присвоємо функцію змінній:

var sum = function(x, y)
{
  return x + y;
}

Щоб викликати функцію, треба просто записати її ім'я та аргументи у дужках, наприклад, так:

a = sum(5, 3);

Масиви

Приклад оголошення порожнього масиву:

var arr = [];

Приклад оголошення ініціалізованого масиву:

var arr2 = ["Arrays", "in", JS];

Звернення до елемента масиву:

var thirdElement = arr[2]; //thirdElement = "JS"

У Javascript, як і у PHP, можна оголосити масив, в якому будуть елементи різних типів:

var arr3 = [2.3, true, 5];

Також можна вставляти елементи у масив не послідовно:

arr3[100] = "legit";

Об'єкти

Щоб створити пустий об'єкт у Javascript, достатньо при оголошенні змінної вказати пусті фігурні дужки:

var emptyObject = {};

Створення нових властивостей, або полів об'єкта, відбувається одним із двох способів:

emptyObject["newProperty"] = "newValue"; //Перший спосіб
emptyObject.otherNewProperty = "otherNewValue"; //Другий спосіб

Доступ до полів об'єкта також можна отримати двома шляхами, як у прикладі:

alert(emptyObject.otherNewProperty);	
alert(emptyObject["otherNewProperty"]);

При оголошенні масива, можна одразу задати у ньому поля:

var CS50 = { 
  <ім'я поля>: <значення>,
  "course": "CS50", 
  "instructor": "David J. Malan '99", 
  "tfs": ["R.J.", "Ben", "Pat", "Chris"], 
  "psets": 8,  
  "taped": true 
};

Приклад створення массиву об'єктів та доступу до цих об'єктів:

var cottage = [ 
  {name: "James", house: "Winthrop"},  
  {name: "Molly", house: "Cabot"}, 
  {name: "Carl", house: "Kirkland"} 
]; 
//Цикл для доступу до елементів масиву
for(var i = 0; i < cottage.length; i++) 
{ 
alert(cottage[i].name); 
}

DOM - Document-Object Model

Document-Object Model (об'єктна модель документа) - це стандартний програмний інтерфейс, що дозволяє отримувати доступ до елементів веб-сторінки, а також змінювати їх вміст. Кожна веб-сторінка може бути представлена у вигляді деревовидної структури:

<!DOCTYPE html> 
<html> 
  <head> 
    title>hello, world</title> 
  </head> 
  <body> 
     hello, world 
  </body> 
</html>

DOM - це спосіб для доступу до об'єктів цієї деревовидної структури. Приклад доступу до елементів вищенаведеної сторінки:

document.title
document.body
document.body.innerHTML

Також з DOM можна отримати доступ до елемента за значеннями його полів Id, Class або Tag, використовуючи відповідну функцію:

document.getElementById(<id елемента>)	
document.getElementsByClassName(<class елемента>)	
document.getElementsByTagName(<Tag елемента>)

Події (Events)

Події у Javascript - це засіб для реагування об'єктів на сторінці на дії користувача. За допомогою подій можна задати, що відбуватиметься при натисканні користувача на кнопку або на зображення.

Приклад роботи з подіями:

//Подія: завантаження сторінки
window.onload = function() { 
  var searchButton =  document.getElementById("search_button"); 
  //Подія: натискання на кнопку
  searchButton.onclick = function() {
      alert("You clicked the search button"); 
  } 
}

У вищенаведеному коді запрограмовано реакцію на дві події: завантаження сторінки та натискання на певну кнопку на сторінці. Схематично це можна зобразити так:

jQuery

jQuery - це корисна бібліотека, що є надбудовою над Javascript. Якщо у Javascript немає якоїсь вбудованої можливості або функції, наймовірніше, що вже є бібліотека, у якій таку функціональність реалізували, а jQuery є однією з найбільших та найпопулярніших бібліотек.

Хоча у цьому семінарі не передбачено розгляд jQuery, рекомендуємо ознайомитись із бібліотекою за допомогою її документації або інших ресурсів.

Приклади

Приклад 1. Вечірка

Створимо просту веб-сторінку, що викорстовує Javascript для зміни фону сторінки.

Код сторінки party.html:

<!doctype html>
<html>
  <head>
    <link rel = "stylesheet" type="text/css" href="party.css">
  </head>
  <body id="background">
        <a id="party" href="#">CLICK HERE TO PARTY</a>
         <script type="text/javascript" src="party.js"> </script>
  </body>
</html>

Код опису стилів party.css:

a {
  font-family: 'Helvetica Nueue', Helvetica, Verdana, sans-serif;
  font-size: 48px;
  font-weight: bold;
  color: blue;
  text-decoration: none;
}

Створимо файл party.js. Для того, щоб змінити колір тла сторінки, необхідно отримати об'єкт, що являє собою тло:

var background = document.getElementById('background');

А також потрібно, щоб якийсь об'єкт реагував на натискання на ньому. Нехай це буде посилання, розташоване на веб-сторінці. Отримаємо цей об'єкт за допомогою DOM:

var button = document.getElementById('party');

Далі, визначимо в обробнику події onclick функцію, яка в майбутньому змінюватиме тло сторінки:

button.onclick = function() {
};

Тепер у тілі цієї функції, оголосимо три змінні, що визначають червоний, зелений і синій компоненти кольору відповідно. Кожну змінну ініціалізуємо випадковим значенням, а результат встановимо у якості тла сторінки:

var red = Math.floor(Math.random() * 255);
var green = Math.floor(Math.random() * 255);
var blue = Math.floor(Math.random() * 255);
background.style.background = 'rgb(' + red + ',' + green + ',' + blue + ')';

Релультом нашої роботи буде файл party.js:

var background = document.getElementById('background');
var button = document.getElementById('party');
button.onclick = function() {
  var red = Math.floor(Math.random() * 255);
  var green = Math.floor(Math.random() * 255);
  var blue = Math.floor(Math.random() * 255);
  background.style.background = 'rgb(' + red + ',' + green + ',' + blue + ')';
};

Демонстрацію роботи прикладу можна подивитись тут.

Приклад 2. Годинник

Створимо сторінку, яка відображатиме поточний час. Вміст файлу clock.html:

<!doctype html>
<html>
    <body>
        <div id="clock"> This should be a clock!</div>
        <script type="text/javascript" src="clock.js"></script>
        </script>
    </body>
</html>

Розробимо скрипт для сторінки, і помістимо його у файл clock.js. Найперше, що треба зробити - це отримати об'єкт DOM, у якому буде відображатись годинник:

var clock = document.getElementById('clock');

Функція setInterval виконує функцію, що передається їй у якості першого аргумента, раз у стільки мілісекунд, скільки передано у якості другого аргумента. Використаємо цю функцію для генерації годинника:

setInterval(function() {
  // Генеруємо годинник тут
}, 100)

У тілі функції для генерації годинника встановимо у якості тесту елемента clock поточний час, використавши системний об'єкт Date:

var date = new Date();
var html = '';
html += '<h1>' + date.getHours() + ':' + date.getMinutes() + ':' + date.getSeconds() + '</h1>';
clock.innerHTML = html;

Результатом роботи буде файл clock.js:

var clock = document.getElementById('clock');
setInterval(function() {
  // Генеруємо годинник тут
  var date = new Date();
  var html = '';
  html += '<h1>' + date.getHours() + ':' + date.getMinutes() + ':' + date.getSeconds() + '</h1>';
  clock.innerHTML = html;
}, 100)

Приклад 3. Twitter

Багато хто сьогодні користується сервісом Twitter, у якому є обмеження на кількість симолів у записах. За допомогою Javascript ми можемо створити схоже на Twitter поле вводу з обмеженням на кількість символів.

Для початку створимо html-сторінку з заготовкою (twitter.html):

<!doctype html>
<html>
 <head>
 </head>
 <body>
  <textarea id="text" style="width: 500px; height: 150px;"></textarea>
  <h1>Characters left: <span id="characters-remaining">140</span>
  <script type="text/javascript" src="twitter.js"> </script>
 </body>
</html>

Створимо файл twitter.js, у якому буде міститись код, який накладає обмеження на кількість символів. Отримаємо об'єкти поля введення та кількості символів, які лишилсь у користувача. Створимо обробник події натискання на клавішу у полі введення:

var textarea = document.getElementById('text');
var remaining = document.getElementById('characters-remaining');
//Обробник натискання
textarea.onkeypress = function(e) {
  //Встановимо обмеження тут
};

В обробнику обмеження, отримаємо кількість символів, які ввів користувач, обчислимо кількість символів, які залишились та покажемо їх на веб-сторінці. Якщо у користувача не залишилось символів, то повернемо результат false - це попередить появу нових символів у полі вводу:

var r = 140 - textarea.value.length;
remaining.innerHTML = r;
if(r <= 0) {
  return false;
}

Результатом роботи є файл twitter.js:

var textarea = document.getElementById('text');
var remaining = document.getElementById('characters-remaining');

textarea.onkeypress = function(e) {
  var r = 140 - textarea.value.length;
  remaining.innerHTML = r;
  if(r <= 0) {
    return false;
  }
};

Демонстрацію роботи прикладу можна подивитись тут.
