Питання 1. Стек та купа

Розкажіть, будь ласка, ще раз про стек, купу та принцип їх роботи.

Відповідь

На малюнку нижче показано загальну модель організації пам'яті комп'ютера.

Стек - це область пам'яті, над якою ви не маєте ніякого контролю. У нього записуються змінні та інофрмація, що створюється у результаті виклику будь-яких функцій. Коли функція закінчує роботу, то вся інформація про її виклик та її змінні видаляються з стеку автоматично.

Купа - це та область пам'яті, над якою ви, як програмісти, маєте безпосередній контроль. Пам'ять тут виділяється у результаті виклику функції malloc.

Глибші знання про стек та купу вам поки що не знадобляться. Ви їх отримаєте пізніше, у поглиблених курсах.

Питання 2. Переповнення буфера

Що таке "переповнення буфера" (buffer overflow)?

Відповідь

Отже, що таке буфери? Буфер - це масив певної довжини, розташований у пам'яті. Переповнення буферу виникає, якщо ми намагаємось записати в нього більше даних, аніж передбачено розміром буфера. За допомогою переповнення буфера зловмисник може записати небезпечний код у пам'ять вашого комп'ютера.

Питання 3. Покажчики

Чи могли б ви ще раз повторити про покажчики? Про базовий синтаксис.

Відповідь

Щоб створити покажчик, необхідно вказати тип даних, на який посилатиметься покажчик, записати зірочку та ім'я покажчика. Наприклад, так:

int* a;     //Покажчик на int
char* b;    //Покажчик на char
float* c;   //Покажчик на float

Для того, щоб отримати значення, на яке вказує покажчик, необхідно застосувати оператор розіменування:

*<ім'я змінної>

Щоб отримати покажчик на якусь змінну, треба використати оператор взяття адреси:

&<ім'я змінної>

Питання 4. Структури та списки

Хотілося б освіжити в пам'яті зання про структури (stucts) та зв'язані списки.

Відповідь

Отже, структури (stucts) - це просто засіб, щоб в одному місці зберігати дані кількох різних типів. У масивах можна зберігати лише цілі числа, або лише символи, - загалом, дані лише одного типу.

Структури - це ваш особистий тип даних, в який входять елементи різних типів даних. Приклад оголошення структури, в якій зберігаються дані про студента:

typedef struct
{
  string name;
  int id;
  int phone_number;
  string house;
}
student;

Тепер у нас є тип із іменем student. Ми можемо створювати змінні цього типу, і використовувати поля цього типу як показано нижче:

student walker; //Створення нової змінної

//Заповнення полів нової змінної
walker.name = "Walker";
walker.id = 123456;
walker.phone_number = 38012312366;
walker.house = "Jackson"
