PHP

Що таке PHP?

PHP (розшифровується як PHP Hypertext Preprocessor - гіпертекстовий препроцессор PHP) - це мова програмування, що використовується на сервері. Завдяки можливості комбінувати код PHP та HTML, можна створювати динамічні веб-сторінки, що показують кожного разу різну інформацію в залежності від, наприклад, налаштувань користувача.

Швидкий огляд мови PHP

Змінні. Оголошення змінних

На відміну від С, PHP - мова з динамічною типізацією. Це означає, що вам не треба вказувати тип змінної при її оголошенні - він визначається автоматично.

Синтаксис оголошення змінної:

$змінна = значення;

Наприклад:

$var = 3;

Зверніть увагу, що імена усіх змінних в PHP починаються зі знаку $

Масиви

Масиви у PHP можуть бути асоціативними - тобто індексом може бути не лише число, а й якийсь об'єкт (наприклад, рядок). Синтаксис оголошення масиву:

$array_name = [key1 => value1, key2 => value2]

Тоді, щоб звернутись до елементу масиву, необхідно застосувати таку конструкцію:

$array_name[key1];

При оголошенні масиву, ключі можна не вказувати, і тоді індексами в масиві будуть числа, так само як і в C:

$array_2 = [10, 20, 30];
$array_2[1] = 20;

Приклад роботи з асоціативним масивом (команда echo виводить текст у командний рядок):

<?php
$array = ["Allison" => "Buchholtz-Au", "Ema" =>"Horvath"];
echo "{$array['Ema']}\n";
?>

Оператори порівняння

Так як мова PHP має динамічну типізацію, у ній існує два оператори порівняння: == та ===. Оператор == спершу здійснює приведення типів (перетворює порівнювані змінні до одного типу даних), а потім порівнює їх значення. Оператор === перевіряє чи обидва порівнювані елементи мають однаковий тип та значення.

Цикл foreach

Цикл foreach - це зручний спосіб проходження по масиву. Замість створення спеціальних тимчасових змінних для проходження по масиву, достатно просто написати:

foreach	($array	as $value) {	
  //виконати ці інструкції
}

Якщо необхідно пройтися по асоціативному масиву, де є ключі:

foreach	($array	as $key => $value) {	
  //виконати ці інструкції
}

Приклади:

У цьому прикладі відбувається ітерація по всіх елементах масиву, і кожен елемент масиву (і його ключ) виводяться на екран:

<?php
$arr = array("one", "two", "three");
foreach ($arr as $key => $counter) {
  echo "Value:$key => $counter\n";
}?>

У наступному прикладі відбувається те ж, що й у попередньому, але ми використовуємо асоціативний масив. Зверніть увагу, що елементи масиву необов'язково повинні мати однаковий тип:

<?php
$arr = array("a" => 1, "b" => "two", "c" => "three");
foreach ($arr as $key => $value) {
echo "Key: $key; Value: $value\n";
}?>

Динамічна типізація

Ми вже говорили про динамічну типізацію. Щоб краще зрозуміти, що це таке, продемонструємо декілька прикладів.

Приклад 1

У цьому прикладі у нас є змінна $a, якій присвоєно деяке значення. Ми виводимо на екран значення і тип цієї змінної.

<?php
$a = 1 + 1;
echo "1 + 1 is $a\n";
$typea = gettype($a);
echo "Type: $typea \n";?>

?>

Передача даних у PHP

Як ви вже, мабуть, знаєте, у HTTP є два основних способи передачі даних між клієнтом і сервером: методи GET та POST. Метод GET передає дані через URL, а POST - через тіло HTTP-запиту.

У цьому прикладі створено просту форму для введення імені та віку користувача. При натисканні кнопки форма відправляє дані на сервер за допомогою POST-запиту.

<form action="action.php" method="post">
  <p>Your name: <input type="text" name="name"/></p>
  <p>Your age: <input type="text" name="age"/></p>  
  <p>input type="submit"></p>
</form>

Дані, що передаються за допомогою методу POST, у PHP зберігаються у спеціальний асоціативний масив $_POST, де ключами є імена параметрів. Отже, ми можемо отрмати до них доступ наступним чином:

Hi <?php echo htmlspecialchars($_POST['name']); ?>
Your are <?php echo (int)$_POST['age']; ?> years old.

SQL - Structured Query Language

SQL - це мова запитів, що використовується у базах даних для того, щоб записувати або зчитувати інформацію. В SQL є чотири основних типи запитів:

  • UPDATE

    • Оновити вже існуючі дані у базі.

  • INSERT INTO

    • Додати певні значення у базу.

  • SELECT

    • Вибрати і переглянути дані з бази.

  • DELETE

    • Видалити дані з бази.

Приклади

Нехай у нас є база даних, що складається з таблиці data із наступним вмістом:

UPDATE

Нехай ми хочемо змінити дані для запису із іменем Emma. Тоді треба виконати такий запит:

UPDATE data
SET year = 16
WHERE id = 2;

MVC - Model View Controller

MVC (модель, вигляд, контроллер) - це шаблон розробки, що полягає у виділенні окремих елементів програми, що відповідають за модель, відображення та маніпуляцію даними.

Model (модель)

Цей елемент MVC відповідає за зберігання інформації та організацію даних. Це може бути, наприклад, база даних MySQL або файл з даними.

View (вигляд)

Цей елемент MVC відповідає за відображення даних користувачу - тобто, це є інтерфейс користувача. Приклад: HTML, можливо PHP.

Controller (контролер)

Контролер відповідає за обробку запитів від користувача і взаємодіє з моделлю, щоб оновлювати або завантажувати з неї дані. Приклад: PHP.
